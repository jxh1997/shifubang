# shifubang

## 项目简介

本项目主要使用 Vue3 + AntD 2.0 本版进行开发。

主要实现功能：
- 登录页面
- 后台订单管理页面
- 订单的增、删、改、查等操作

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
